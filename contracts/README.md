# Beginner-Friendly NFT Deployment Workflow

This project provides a simple, beginner-friendly workflow for deploying NFTs using Hardhat, React, and VS Code. It includes:
- ERC721 smart contract (OpenZeppelin)
- Automated tests
- Deployment scripts for testnet and mainnet
- Simple React frontend for minting/viewing NFTs

## Project Structure
- `contracts/` — Solidity smart contracts (ERC721)
- `scripts/` — Deployment and minting scripts
- `test/` — Automated contract tests
- `frontend/` — React frontend app

## Excluded (for future releases)
- IPFS integration for metadata/images
- Advanced NFT features (royalties, batch minting, etc.)
- CI/CD automation

See the code and comments for step-by-step instructions. Follow the README for setup and deployment.
